**UnifiedPatent model (Pydantic):**
```python
from typing import Optional
from pydantic import BaseModel, Field
from .legal_status import PatentStatus

class UnifiedPatent(BaseModel):
    """Unified patent model for display from multiple sources."""
    id: str = Field(description="Patent ID (publication number)")
    title: str
    abstract: Optional[str] = None
    date: Optional[str] = None  # Grant or publication date
    source: PatentSource
    url: str = Field(description="Link to patent on official site")

    # Optional fields that may not be available from all sources
    status: Optional[PatentStatus] = None
    cpc_codes: list[str] = Field(default_factory=list)

    class Config:
        extra = "ignore"

    @classmethod
    def from_uspto(cls, patent: "Patent") -> "UnifiedPatent":
        """Convert USPTO Patent to UnifiedPatent."""
        return cls(
            id=f"US{patent.patent_id}",
            title=patent.patent_title,
            abstract=patent.patent_abstract,
            date=str(patent.patent_date) if patent.patent_date else None,
            source=PatentSource.USPTO,
            url=f"https://patents.google.com/patent/US{patent.patent_id}",
        )

    @classmethod
    def from_epo(
        cls,
        patent: "EPOPatent",
        status: Optional[PatentStatus] = None
    ) -> "UnifiedPatent":
        """Convert EPO Patent to UnifiedPatent."""
        return cls(
            id=patent.publication_number,
            title=patent.title or "Untitled",
            abstract=patent.abstract,
            date=patent.publication_date,
            source=PatentSource.EPO,
            url=f"https://worldwide.espacenet.com/patent/search?q=pn%3D{patent.publication_number}",
            status=status,
            cpc_codes=patent.cpc_classifications,
        )
```

Import Patent from .uspto and EPOPatent from .epo within the methods to avoid circular imports.

**perform_epo_search function:**
```python
def perform_epo_search(
    problem: str,
    solution: str,
    constraints: str,
    consumer_key: str,
    consumer_secret: str,
    filter_active_only: bool = True,
    is_cancelled: Callable[[], bool] = lambda: False,
    progress_callback: Optional[Callable[[int, int, str], None]] = None,
) -> EPOSearchResponse:
```

5-step progress reporting:
1. "Extracting keywords..." (20%)
2. "Building EPO query..." (40%)
3. "Searching EPO patents..." (60%)
4. "Filtering active patents..." (80%) - only if filter_active_only=True
5. "Search complete" (100%)

Implementation steps:
1. Extract keywords using extract_search_terms (from keyword_extractor)
2. Build CQL query with cosmetic CPC codes
3. Search using EPOClient.search_patents
4. If filter_active_only:
   - For each patent, call get_legal_status
   - Filter to only patents where is_patent_active(status) is True
   - Note: This is slow for many patents; log warning if >50 patents
5. Return EPOSearchResponse

Handle cancellation: Check is_cancelled() after each step, return empty EPOSearchResponse if cancelled.

Wrap exceptions in EPOSearchError for consistent handling.
