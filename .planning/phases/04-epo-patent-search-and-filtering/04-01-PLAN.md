Create `src/fto_agent/services/epo.py` following the USPTOClient pattern:

**EPOPatent model (Pydantic):**
- publication_number: str (e.g., "EP1000000A1")
- title: Optional[str]
- abstract: Optional[str]
- publication_date: Optional[str]
- applicants: list[str] = []
- cpc_classifications: list[str] = []
- Config: extra = "ignore"

**EPOSearchResponse model (Pydantic):**
- patents: list[EPOPatent] = []
- count: int = 0
- total_hits: int = 0

**EPOSearchError exception:**
- message: str
- status_code: Optional[int]

**EPOClient class:**
- COSMETIC_CPC_CODES = ["A61K8", "A61Q"]
- __init__(consumer_key=None, consumer_secret=None): Read from env vars EPO_OPS_CONSUMER_KEY and EPO_OPS_CONSUMER_SECRET if not provided. Raise EPOSearchError if missing. Create epo_ops.Client(key=key, secret=secret).
- search_patents(keywords, include_cosmetic_cpc=True, range_begin=1, range_end=100) -> EPOSearchResponse: Build CQL query, call client.published_data_search(), parse XML response.
- get_legal_status(publication_number) -> bytes: Call client.legal(reference_type="publication", input=Epodoc(publication_number)), return raw XML response.
- _build_cql_query(keywords, include_cosmetic_cpc) -> str: Build CQL like 'ta="keyword1 keyword2" AND (cpc=A61K8 OR cpc=A61Q)'
- _parse_search_response(response) -> EPOSearchResponse: Parse XML using lxml with namespaces, extract patent data.
- close(), __enter__, __exit__: Context manager support.

**XML Parsing constants:**
```python
NAMESPACES = {
    'ops': 'http://ops.epo.org',
    'exchange': 'http://www.epo.org/exchange',
}
```

Create `src/fto_agent/services/legal_status.py`:

**PatentStatus enum:**
- ACTIVE = "active"
- EXPIRED = "expired"
- LAPSED = "lapsed"
- WITHDRAWN = "withdrawn"
- PENDING = "pending"
- UNKNOWN = "unknown"

**INACTIVE_KEYWORDS constant:**
```python
INACTIVE_KEYWORDS = [
    "lapse", "lapsed", "expired", "withdrawn",
    "refusal", "refused", "revoked", "revocation",
    "abandoned", "non-payment", "not paid",
]
```

**get_patent_status(legal_xml: bytes) -> PatentStatus:**
Parse INPADOC legal status XML. Look for event descriptions containing INACTIVE_KEYWORDS. If recent events indicate lapse/withdrawal/expiry, return appropriate status. Default to UNKNOWN if parsing fails.

**is_patent_active(status: PatentStatus) -> bool:**
Return True if status is ACTIVE, PENDING, or UNKNOWN (err on side of caution).
