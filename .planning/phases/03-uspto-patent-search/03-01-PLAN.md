**Create src/fto_agent/services/uspto.py** with:

1. **Patent** (Pydantic BaseModel):
   - patent_id: str (required)
   - patent_title: str (required)
   - patent_abstract: Optional[str] = None
   - patent_date: Optional[date] = None
   - patent_type: Optional[str] = None
   - Config: extra = "ignore"

2. **PatentSearchResponse** (Pydantic BaseModel):
   - patents: list[Patent] = Field(default_factory=list)
   - count: int = Field(default=0)
   - total_hits: int = Field(default=0)
   - Config: extra = "ignore"

3. **USPTOSearchError** (Exception):
   - message: str
   - status_code: Optional[int] = None

4. **USPTOClient** class:
   - BASE_URL = "https://search.patentsview.org/api/v1"
   - DEFAULT_FIELDS = ["patent_id", "patent_title", "patent_abstract", "patent_date", "patent_type"]
   - __init__(self, api_key: Optional[str] = None, timeout: float = 30.0)
     - If api_key is None, read from PATENTSVIEW_API_KEY env var
     - Raise USPTOSearchError if no API key available
     - Create httpx.Client with base_url, X-Api-Key header, timeout
   - search_patents(self, query: dict, fields: Optional[list[str]] = None, size: int = 100, sort: Optional[list[dict]] = None) -> PatentSearchResponse
     - Clamp size to 1-1000
     - POST to /patent/ with JSON body: q, f, o (options), s (sort)
     - Handle 429 rate limit error with Retry-After header
     - Raise USPTOSearchError on any error
     - Return PatentSearchResponse.model_validate(data)
   - close(self), __enter__, __exit__ for context manager support

5. **build_keyword_query(keywords: list[str]) -> dict**:
   - Raise ValueError if keywords is empty
   - Join keywords with space
   - Return: {"_or": [{"_text_any": {"patent_title": keyword_string}}, {"_text_any": {"patent_abstract": keyword_string}}]}

Follow the exact patterns from 03-RESEARCH.md. Use comprehensive docstrings.

   - Deduplicate while maintaining priority order
   - Return first max_total keywords
