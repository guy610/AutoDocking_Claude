  <action>
Create src/fto_agent/workers/base.py with two classes:

1. WorkerSignals(QObject):
   - Signal: started = Signal()
   - Signal: progress = Signal(int, int, str)  # current, total, message
   - Signal: finished = Signal()
   - Signal: result = Signal(object)
   - Signal: error = Signal(tuple)  # (exctype, value, traceback_str)
   - Include docstring explaining each signal

2. Worker(QRunnable):
   - __init__(self, fn, *args, **kwargs): Store function, args, kwargs; create WorkerSignals instance; set _is_cancelled = False
   - cancel(self): Set _is_cancelled = True
   - is_cancelled property: Return _is_cancelled
   - run(self) with @Slot() decorator:
     * Emit started signal
     * Try: Call fn(*args, is_cancelled=lambda: self._is_cancelled, progress_callback=self.signals.progress.emit, **kwargs)
     * Except: Emit error signal with (exctype, value, traceback.format_exc())
     * Else (not cancelled): Emit result signal
     * Finally: Emit finished signal

Create src/fto_agent/workers/__init__.py:
- Export Worker and WorkerSignals from base.py

Pattern source: Research document Pattern 2 (Cancellable Worker).
Do NOT use QThread.terminate() - use cooperative cancellation only.

  </verify>
